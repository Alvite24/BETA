# 🖥️ PCBuilder Pro

**Configurador inteligente de PCs con recomendaciones personalizadas**

Una aplicación web interactiva que ayuda a los usuarios a encontrar la configuración de PC perfecta según sus necesidades y presupuesto, con enlaces directos de compra a Amazon y PcComponentes.

---

## ✨ Características

- 🎯 **Recomendaciones personalizadas** basadas en el uso (Gaming, Edición, Oficina, Programación)
- 💰 **Tres niveles de presupuesto** (Bajo, Medio, Alto)
- 🛒 **Enlaces directos de compra** a Amazon y PcComponentes
- 🌓 **Modo oscuro/claro** con almacenamiento de preferencias
- 📱 **Diseño responsive** - funciona en móviles, tablets y escritorio
- 🎨 **Interfaz moderna** con animaciones suaves
- ⚡ **Rápido y ligero** - HTML, CSS y JavaScript vanilla

---

## 🔧 Componentes Recomendados

Cada configuración incluye:
- ✅ Procesador (CPU)
- ✅ Tarjeta Gráfica (GPU)
- ✅ Memoria RAM
- ✅ Almacenamiento (SSD/NVMe)
- ✅ Caja del PC
- ✅ Fuente de Alimentación (PSU)

---

## 🛠️ Tecnologías Utilizadas

- **HTML5** - Estructura semántica
- **CSS3** - Diseño moderno con variables CSS y flexbox/grid
- **JavaScript (ES6+)** - Lógica de la aplicación
- **Font Awesome** - Iconos
- **Google Fonts** - Tipografía (Inter)

---

## 🚀 Cómo Usar

1. **Clona el repositorio:**
   ```bash
   git clone https://github.com/Alvite24/pcbuilder-pro.git
   cd pcbuilder-pro
   ```

2. **Abre el archivo:**
   - Simplemente abre `index.html` en tu navegador

3. **O usa un servidor local:**
   ```bash
   # Con Python
   python -m http.server 8000
   
   # Con Node.js
   npx serve
   ```

4. **Accede en tu navegador:**
   ```
   http://localhost:8000
   ```
