# 🛡️ PCBuilder Pro - Mantenimiento y Cuidados

**La guía definitiva para entender y cuidar tu hardware**

Una extensión de la plataforma **PCBuilder Pro** diseñada específicamente para enseñar a los usuarios cómo mantener sus equipos informáticos en óptimas condiciones físicas y lógicas, maximizando su rendimiento y vida útil.

---

## ✨ Características de la Guía

Esta plataforma incluye guías detalladas sobre 6 pilares fundamentales del cuidado de un PC:

- 🧹 **Limpieza Física:** Técnicas y herramientas para eliminar el polvo y cuidar tus ventiladores.
- ⚙️ **Optimización de Sistema:** Cómo limpiar tu SO (Windows) de bloatware y actualizar drivers.
- 🌡️ **Control Térmico:** Guía de monitorización de temperaturas y rangos seguros para CPU/GPU.
- 🔌 **Cable Management:** La importancia de esconder los cables para un flujo de aire perfecto.
- 🚀 **Actualizaciones (Upgrades):** Cómo identificar cuellos de botella y saber cuándo poner más RAM o un SSD nuevo.
- 💾 **Copias de Seguridad:** Implementación de la estrategia 3-2-1 para proteger tus datos de ransomware o fallos físicos.

---

## 🎨 Diseño y UI

- 🌓 **Tema Oscuro Moderno** con acentos neón y estilo "cyberpunk"
- 📱 **100% Responsive** - Experiencia fluida en teléfonos móviles, tablets y monitores grandes
- ✨ **Animaciones CSS** en tarjetas e iconos
- 🧭 **Barra de navegación "Sticky"** con difuminado (backdrop-filter)
- 🚀 **Navegación suave** (Smooth scrolling) con JavaScript Vanilla

---

## 🛠️ Tecnologías Empleadas

- **HTML5** - Estructura semántica moderna (`<article>`, `<section>`, `<nav>`, etc.)
- **CSS3** - Uso avanzado de variables CSS, Flexbox, CSS Grid y filtros de sombra
- **JavaScript (ES6+)** - Lógica del menú hamburguesa y navegación interactiva sin dependencias
- **Font Awesome 6** - Iconografía vectorial
- **Google Fonts** - Tipografías *Outfit* (Títulos) e *Inter* (Textos)

## 🚀 Cómo Usar / Despliegue Local

1. **Clona el repositorio:**
   ```bash
   git clone https://github.com/Alvite24/Pagina-web-Prueba-2.git
   cd Pagina-web-Prueba-2
   ```

2. **Abre el proyecto:**
   - El proyecto no requiere compilación. Simplemente abre el archivo `index.html` haciendo doble clic sobre él en tu explorador de archivos y se abrirá en tu navegador predeterminado.

3. **O usa un servidor local (Opcional):**
   ```bash
   # Con Python (si lo tienes instalado)
   python -m http.server 8000
   
   # Con Node.js/npm (si lo tienes instalado)
   npx serve
   ```
